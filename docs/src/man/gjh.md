# `gradient`, `jacobian`, `hessian`
*Extracts specific partial derivatives already contained in a TPS*
## Syntax
```
grad = gradient(tps)
gradient!(grad, tps)

J = jacobian(map)
jacobian!(J, map)

H = hessian(tps)
hessian!(H, tps)
```
